Team Task Manager Full Stack Project

Tech Stack:
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB

Features:
- User Signup/Login
- JWT Authentication
- Role-based Access (Admin/Member)
- Project Management
- Task Assignment
- Dashboard Statistics

Setup:
1. Open backend folder and run: npm install
2. Open frontend folder and run: npm install
3. Copy .env.example to .env and add your values
4. Start backend: npm run dev
5. Start frontend: npm start
